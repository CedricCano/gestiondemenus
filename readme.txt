Acerca del proyecto:

Este proyecto permite administrar una tabla de mySQL donde se almacenan las opciones de un menù que se presenta dentro del sitio web
asì como darle mantenimiento, permitiendo eliminar, borrar o actualizar esas opciones del menù.

Se ejemplifica còmo se puede realizar un CRUD (Alta baja cambio y consultas de una entidad) utilizando
el modelo vista controlador con PHP interactuando con una base local que viene el script junto con el còdigo fuente dentro de este 
repositorio pùblico para su anàlisis.

<<script_db.sql>> dentro de la carpeta basedatos
